## 🚀 Key Features

*   **⚡ Real-Time JS Rendering:** We render the full DOM (like a real user's browser). We detect links loaded via JavaScript, React, Vue, or lazy-loading widgets.
*   **🔍 Detailed Link Analysis:**
    *   **Status:** Found / Not Found / Donor 404 / Connection Error.
    *   **Attributes:** Checks `dofollow`, `nofollow`, `ugc`, `sponsored`.
    *   **Anchors:** Extracts exact anchor text (Text) or `alt` tags (Images).
*   **🛡️ Tier 2 & PBN Audits:** Ideal for verifying huge lists of guest posts, crowd links, and tiered link building campaigns.
*   **🤖 Indexability Check:** Optionally checks if the donor page itself is indexed by Google (preventing "ghost links").
*   **💰 Cost-Effective:** Pay-as-you-go model starting from **$0.0007 per check**.

## 📦 API Usage Example

Full documentation: [SpeedyIndex API Docs](https://en.speedyindex.com/api.php)

### 1. Create a Check Task

**Endpoint:** `POST https://api.speedyindex.com/v2/task/backlink/create`

```bash
curl -X POST "https://api.speedyindex.com/v2/task/backlink/create" \
     -H "Authorization: Bearer YOUR_API_KEY" \
     -H "Content-Type: application/json" \
     -d '{
           "urls": [
             "https://donor-site.com/article-1",
             "https://donor-site.com/article-2"
           ],
           "target": "your-money-site.com"
         }'
```

### 2. Get Results

**Response Example:**

```json
{
  "status": "completed",
  "data": [
    {
      "donor_url": "https://donor-site.com/article-1",
      "link_status": "found",
      "anchor": "best seo tools",
      "rel": "dofollow",
      "link_type": "text",
      "is_indexed": true
    },
    {
      "donor_url": "https://donor-site.com/article-2",
      "link_status": "not_found",
      "error": "link_removed"
    }
  ]
}
```
